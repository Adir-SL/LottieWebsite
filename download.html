Installation

  1. Download the latest release file
  2. Extract the ZIP file
  3. Copy the APP file to your Application folder
  4. Double click to open Lottie Player
  5. If it does not open for security reasons, click Done
  6. Go to System Settings then click Privacy & Security, then click Open and Open Anyway
