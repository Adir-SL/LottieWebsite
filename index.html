Lottie Player

It plays Lottie Animations.
On your own machine.

DOWNLOAD FOR MAC DOWNLOAD FOR WINDOWS

Features

  • View and Play Lottie Animations (.json files)
  • Synchronizing two different animations
  • Play animations in Full Screen or Side by Side
  • Click an animation to Pause or Play it
  • Playing animations in Loop or Play Once

Installation

  1. Download the latest release file
  2. Extract the ZIP file
  3. Copy the APP file to your Application folder
  4. Double click to open Lottie Player
  5. If it does not open for security reasons, click Done
  6. Go to System Settings then click Privacy & Security, then click Open and Open Anyway
